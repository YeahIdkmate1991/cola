There is also the unprotected copy of this cast file (CST) along with the source code dumped so that you can see Amy's custom Lingo (she essentially said fuck Flash Remoting and copied Habbo's architecture lol)

Originals - Contains a set of Coke Studios client files from March 2006. The files have been renamed to the proper names that they orignally had (found within the client source code), the public rooms placed in a subdirectory (which is where the client looks for them), and there is a loader client.php which should work for launching the client in a web browser. The assets subdirectory contains uneditted versions of SF_Gateway.cct/swf

SF_Gateway Localhost Edit - Inside of the client files there is a cast named SF_Gateway.cct which contains a cast member titled SF_Gateway.swf inside of this SWF is the 'core' of the Coke Studios client logic. This file has hardcoded URIs which were used to contact servers, load assets, etc. In order to get the original client working on a localhost environment the SWF must be editted in JPEXS and then must be re-added to the CCT SF_Gateway. Essentially this directory contains the CCT and unprotected CST along with a pre-editted copy of the SWF which should work on a localhost environment.
